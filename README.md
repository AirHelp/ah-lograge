# Ah::Lograge
Common initializer for Lograge across Airhelp Rails projects

## Usage
just install it. You can pass env variable `DISABLE_LOGRAGE` to disable it.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ah-lograge'
```

And then execute:
```bash
$ bundle
```
